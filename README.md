# Unzipper Pro

A professional native archive extractor for macOS supporting all major formats.

## Download

Download the latest version from the [Releases](https://github.com/NULLSHADExXx/unzipper-pro/releases) page.

| Platform | File |
|----------|------|
| macOS 14+ | [UnzipperPro.dmg](https://github.com/NULLSHADExXx/unzipper-pro/releases/latest/download/UnzipperPro.dmg) |

## Features

- Extract ZIP, RAR, 7z, TAR, and more
- Drag-and-drop extraction
- Password-protected archives
- Native macOS design

## Installation

1. Download `UnzipperPro.dmg` from the latest release
2. Open the DMG file
3. Drag the app to your Applications folder
4. Launch from Applications

### macOS Gatekeeper

Since Unzipper Pro is distributed outside the App Store, macOS may show a warning. To resolve:

```bash
xxattr -cr /Applications/Unzipper\ Pro.appp
```

Then launch the app again.

## Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon or Intel Mac
